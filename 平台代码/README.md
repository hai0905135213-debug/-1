# 校园饭搭子平台

这是校园饭搭子智能匹配平台的代码目录。

第一阶段目标是先跑通最小闭环：

1. 用户登录并完善基础资料
2. 用户发起一个饭局
3. 其他用户浏览并加入饭局
4. 人数满足后成局
5. 线下就餐后进行简单评价

## 第一版范围

第一版只做拼餐撮合工具，不做完整社交平台。

核心页面：

- 首页：饭局列表、快速发起入口
- 发起饭局页：填写吃什么、时间、地点、人数、预算、聊天模式
- 饭局详情页：查看信息并加入/退出
- 我的饭局页：查看我发起和我加入的饭局
- 我的资料页：学生信息、口味偏好、信用分

核心后端模块：

- 用户模块
- 资料模块
- 饭局模块
- 参与者模块
- 评价与信用模块

## 建议技术栈

- 前端：微信小程序或 uni-app
- 后端：Node.js + Express
- 数据库：MySQL
- 部署：腾讯云/阿里云轻量服务器
