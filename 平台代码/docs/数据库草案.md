# 数据库草案

数据库暂定使用 MySQL。

## `users` 用户表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | BIGINT | 用户 ID |
| `openid` | VARCHAR(128) | 微信 openid |
| `nickname` | VARCHAR(64) | 昵称 |
| `avatar_url` | VARCHAR(255) | 头像 |
| `school` | VARCHAR(128) | 学校 |
| `credit_score` | INT | 信用分 |
| `created_at` | DATETIME | 创建时间 |
| `updated_at` | DATETIME | 更新时间 |

## `profiles` 用户资料表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | BIGINT | 资料 ID |
| `user_id` | BIGINT | 用户 ID |
| `verified_student` | TINYINT | 是否学生认证 |
| `taste_tags` | JSON | 口味标签 |
| `chat_style` | VARCHAR(64) | 聊天模式 |
| `bio` | VARCHAR(255) | 简介 |
| `created_at` | DATETIME | 创建时间 |
| `updated_at` | DATETIME | 更新时间 |

## `meal_posts` 饭局表

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | BIGINT | 饭局 ID |
| `host_user_id` | BIGINT | 发起人 ID |
| `title` | VARCHAR(100) | 饭局标题 |
| `meal_time` | DATETIME | 饭点时间 |
| `location` | VARCHAR(128) | 地点 |
| `restaurant_id` | BIGINT | 餐厅或食堂窗口 ID，可为空 |
| `max_people` | INT | 最大人数 |
| `budget_min` | DECIMAL(10,2) | 最低预算 |
| `budget_max` | DECIMAL(10,2) | 最高预算 |
| `chat_mode` | VARCHAR(64) | 聊天模式 |
| `note` | TEXT | 补充说明 |
| `status` | VARCHAR(32) | 状态：open/matched/cancelled/finished |
| `created_at` | DATETIME | 创建时间 |
| `updated_at` | DATETIME | 更新时间 |

## `chat_message_samples` 聊天记录样本表

用于保存脱敏、清洗后的群聊样本，不保存原始可识别个人身份内容。

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `id` | BIGINT | 样本 ID |
| `source_group` | VARCHAR(128) | 来源群标识，需脱敏 |
| `message_time` | DATETIME | 发言时间 |
| `speaker_hash` | VARCHAR(128) | 发言人匿名哈希 |
| `clean_text` | TEXT | 清洗后的文本 |
| `intent` | VARCHAR(64) | 意图：约饭/询问/响应/取消/闲聊 |
| `meal_time_text` | VARCHAR(128) | 文本里的饭点时间 |
| `location_text` | VARCHAR(128) | 文本里的地点 |
| `people_count` | INT | 人数 |
| `budget_text` | VARCHAR(64) | 预算文本 |
| `taste_tags` | JSON | 口味标签 |
| `constraints` | JSON | 忌口、性别、时间等限制 |
| `matched_result` | VARCHAR(32) | 是否成局：unknown/success/failed |
| `failure_reason` | VARCHAR(128) | 未成局原因 |
| `created_at` | DATETIME | 创建时间 |
